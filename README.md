# Siigure Website

## Overview

Siigure Website is a responsive multi-page personal/project website built with **Next.js** and **TailwindCSS**, featuring dynamic content loading and interactive UI components. Designed to work seamlessly on both desktop and mobile devices.

## Features

- Fully responsive layout for desktop and mobile
- Smooth client-side navigation between pages
- Dynamic content rendering for blog/text sections
- Interactive UI elements (buttons, image galleries)
- Optimized performance and image loading

## Tech Stack

- Frontend: Next.js, React, TailwindCSS
- Deployment: Vercel / Netlify
- Version control: Git & GitHub

## Challenges & Learnings

- Resolved layout overflow issues on various screen sizes
- Managed client-side rendering quirks in Next.js
- Improved website performance through image optimization and component structuring
- Gained experience in scalable React component architecture
